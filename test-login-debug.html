❌ Common Error Patterns

1. Invalid Credentials:

❌ Auth Provider: Supabase auth error: AuthApiError: Invalid login credentials

2. No User Profile:

❌ Auth Provider: No profile found, signing out ❌ Login error details: Error: Account not found. Please sign up first.

3. Database Connection Error:

❌ Auth Provider: Database error during login: [database error details]

4. Refresh Token Error:

❌ Auth Provider: Supabase auth error: AuthApiError: Invalid Refresh Token

🛠️ Quick Fixes

If you see "Invalid login credentials":

  • Verify the email and password are correct
  • Check if the user exists in auth.users table
  • Ensure the user's email is confirmed

If you see "Account not found":

  • User exists in auth.users but not in user_profiles table
  • Run the database trigger to create the profile
  • Check the user_profiles table for the email

If you see refresh token errors:

  • Clear browser storage (localStorage and sessionStorage)
  • Try in incognito mode
  • Check for corrupted auth tokens
